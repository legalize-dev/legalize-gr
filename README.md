# Legalize GR

### Ελληνική νομοθεσία σε Markdown, με ιστορικό εκδόσεων στο Git.

Κάθε νόμος είναι ένα αρχείο. Κάθε νομοθετική πράξη του ΦΕΚ Α' είναι ένα commit.

**Επίσημες πηγές:**
- [Internet Archive — Greek Government Gazette](https://archive.org/details/greekgovernmentgazette) (αντίγραφο 1976–2019, μεταφορτωμένο από το Τμήμα Πληροφορικής του Οικονομικού Πανεπιστημίου Αθηνών, `dds@aueb.gr`)
- [search.et.gr](https://search.et.gr) — Εθνικό Τυπογραφείο, για ημερήσιες ενημερώσεις από το 2020 και μετά

Μέρος του έργου [Legalize](https://github.com/legalize-dev/legalize) · [legalize.dev](https://legalize.dev)

> **Αρχική φάση** — Το αποθετήριο βρίσκεται σε ενεργή ανάπτυξη. Η δομή των αρχείων, το ιστορικό των commits και το περιεχόμενο μπορούν να αλλάξουν σημαντικά, συμπεριλαμβανομένων πλήρων αναγεννήσεων.

## Πεδίο εφαρμογής

Αυτή η φάση καλύπτει το **Τεύχος Α' της Εφημερίδας της Κυβερνήσεως** (ΦΕΚ Α') από το **2000 έως το 2019** — δηλαδή νόμους, προεδρικά διατάγματα, πράξεις νομοθετικού περιεχομένου, ψηφίσματα της Αναθεωρητικής Βουλής και άλλες πράξεις του Α' Τεύχους που έχουν δημοσιευθεί ως ψηφιακά PDF (όχι σαρωμένα).

| Έτος | Διαθέσιμο | Σημείωση |
|---|---|---|
| 2000–2019 | ✅ Υποστηριζόμενο | Bootstrap από Internet Archive |
| 2020 και μετά | ⏳ Υπό υλοποίηση | Daily incremental από `searchetv99` |
| Πριν από το 2000 | ⏸️ Αναβολή | Σαρωμένα PDF χωρίς επίπεδο κειμένου — απαιτείται OCR (Phase 3) |

Άλλα τεύχη (Β', Γ', Δ', ΥΟΔΔ, ΑΑΠ, κ.λπ.) δεν περιλαμβάνονται σε αυτή τη φάση.

## Γρήγορη εκκίνηση

```bash
# Κλωνοποίηση
git clone https://github.com/legalize-dev/legalize-gr.git

# Εύρεση νόμου με βάση τον αριθμό ΦΕΚ (έτος + αριθμός φύλλου)
ls gr/GreekGovernmentGazette-2013010*.md

# Ανάγνωση του Κώδικα Φορολογίας Εισοδήματος (Ν. 4172/2013, ΦΕΚ Α' 167/2013)
less gr/GreekGovernmentGazette-20130100167.md

# Ιστορικό μεταβολών ενός νόμου (στην αρχική φάση: ένα bootstrap commit ανά νόμο)
git log --oneline -- gr/GreekGovernmentGazette-20130100167.md

# Αναζήτηση όλων των αναφορών σε προηγούμενο νόμο
grep -l "amends:.*nomos:4046/2012" gr/*.md
```

## Δομή

```
gr/
  GreekGovernmentGazette-YYYYVVNNNNN.md
```

όπου:
- `YYYY` — έτος δημοσίευσης (2000–2019)
- `VV` — τεύχος (`01` = Α')
- `NNNNN` — αριθμός φύλλου, μηδενικά μπροστά μέχρι τα 5 ψηφία

Η δομή είναι **επίπεδη** — ένας μόνο φάκελος, χωρίς υποφακέλους ανά κατηγορία. Ο τύπος της νομοθετικής πράξης βρίσκεται στο πεδίο `rank` του YAML frontmatter:

| `rank` | Τύπος |
|---|---|
| `nomos` | Νόμος ψηφισμένος από τη Βουλή |
| `proedriko_diatagma` | Προεδρικό Διάταγμα (Π.Δ.) |
| `pnp` | Πράξη Νομοθετικού Περιεχομένου (άρθρο 44 §1 του Συντάγματος) |
| `syntagma` | Σύνταγμα και Ψηφίσματα Αναθεωρητικής Βουλής |
| `apofasi_vouli` | Απόφαση Ολομέλειας της Βουλής |
| `praxi_ypourgikou_symvouliou` | Πράξη του Υπουργικού Συμβουλίου |
| `kanonismos` | Κανονισμός (π.χ. της Ιεράς Συνόδου) |
| `apofasi` | Υπουργική / Κοινή Υπουργική Απόφαση που δημοσιεύτηκε στο ΦΕΚ Α' |

## Μορφή αρχείου

Κάθε αρχείο `.md` περιέχει:

```yaml
---
title: "ΝΟΜΟΣ ΥΠ' ΑΡΙΘ. 4172 Φορολογία εισοδήματος, ..."
identifier: "GreekGovernmentGazette-20130100167"
country: "gr"
rank: "nomos"
publication_date: "2013-07-23"
last_updated: "2013-07-23"
status: "in_force"
source: "https://archive.org/details/GreekGovernmentGazette-20130100167"
amends: "nomos:4046/2012,nomos:4093/2012,nomos:4127/2013,..."
---

# ΝΟΜΟΣ ΥΠ' ΑΡΙΘ. 4172 ...

## ΜΕΡΟΣ ΠΡΩΤΟ
### ΤΜΗΜΑ ΠΡΩΤΟ
##### Άρθρο 1
1. Ο Κώδικας Φορολογίας Εισοδήματος (Κ.Φ.Ε.) ρυθμίζει ...
- α) των φυσικών προσώπων,
- β) των νομικών προσώπων ...

##### Άρθρο 2
...
```

Πίνακες (π.χ. φορολογικές κλίμακες) εξάγονται ως Markdown pipe tables:

```markdown
| Φορολογητέο εισόδημα (Ευρώ) | Φορολογικός συντελεστής (%) |
| --- | --- |
| ≤ 25.000 | 22% |
| 25.000,01 έως και 42.000 | 32% |
| > 42.000 | 42% |
```

## Πώς δημιουργείται

Αυτό το αποθετήριο παράγεται αυτόματα από τον [legalize-pipeline](https://github.com/legalize-dev/legalize-pipeline), έναν Python pipeline που:

1. Κατεβάζει τα PDF του ΦΕΚ Α' από το Internet Archive (`pypdfium2` για το κείμενο, `pdfplumber` για τους πίνακες)
2. Καθαρίζει το κείμενο (κεφαλίδα/υποσέλιδο, ενώσεις λέξεων με παύλα, αποκωδικοποίηση Windows-1253 για παλιά PDF)
3. Αναγνωρίζει δομή (άρθρα, κεφάλαια, μέρη, πίνακες, υπογραφές)
4. Παράγει YAML frontmatter με rank, ημερομηνία, παραπομπές σε άλλους νόμους
5. Δημιουργεί ένα git commit ανά νόμο με ιστορική ημερομηνία

**Μηδέν AI/LLM στη διαδικασία.** Όλη η εξαγωγή είναι ντετερμινιστικός κώδικας (regex, parsers, transformers). Η νομοθεσία είναι το επίσημο αρχείο — δεν μπορεί να ερμηνευθεί ή να συμπληρωθεί από ένα μοντέλο.

## Νομικό καθεστώς

Σύμφωνα με τον [Νόμο 4727/2020](https://www.kodiko.gr/nomothesia/document/635566/nomos-4727-2020) (που μεταφέρει την Οδηγία (ΕΕ) 2019/1024 για τα ανοικτά δεδομένα στην ελληνική έννομη τάξη), τα έγγραφα του δημοσίου τομέα — συμπεριλαμβανομένου του ΦΕΚ — είναι ελεύθερα διαθέσιμα προς **εμπορική και μη εμπορική επαναχρησιμοποίηση**.

Το πρωτότυπο περιεχόμενο των ΦΕΚ είναι έργο της Ελληνικής Δημοκρατίας και δημόσιο κτήμα. Η μετατροπή σε Markdown και η οργάνωση σε αυτό το αποθετήριο διατίθενται με την άδεια **MIT** (βλ. [LICENSE](LICENSE)).
